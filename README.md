# Grenade Brothers
## About The Game
- A recreation of the "*Grenade Brothers*" minigame from *Hazelight's* game titled "*A Way Out*"
- Created by Daniel Bullock @DanBullockCS

## How to Play
- Player 1: **A** and **D** to move, **W** to Jump
- Player 2: **Arrow Keys** to Move and Jump
